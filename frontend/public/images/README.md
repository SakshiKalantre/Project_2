# Image Assets for PrepSphere

## How to Add Your Images

1. **College Logo**: 
   - Save your college logo as `spnd-logo.png` in the `logos` folder
   - Recommended size: 100x100 pixels or larger
   - Format: PNG with transparent background preferred

2. **College Building Image**:
   - Save your college building photo as `college-building.jpg` in this directory
   - Recommended size: 1200x600 pixels or larger
   - Format: JPG

## Fallback System

The website has a built-in fallback system:
- If images are missing, colored placeholders with text will be displayed
- The logo placeholder shows "SPND" in white on a maroon background
- The building placeholder shows the college name on a maroon gradient background

## Supported Image Formats

- Logos: PNG, JPG, SVG
- Photos: JPG, PNG, WebP

After adding your images, refresh the browser to see them in place of the placeholders.
